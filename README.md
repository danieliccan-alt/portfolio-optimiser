# Portfolio Optimiser
This project encapsulates live data from 2023 to 2024 using the yfinance library and aims to find the optimal investment for clients across five companies: AAPL, AMZN, JPM, NVDA and TSLA. 

To begin this project, I fetched data using the yfinance library across a two-year scheme (2023-2024) to make my data more readings. The main aim of this project includes being able to calculate daily returns, volatility and Sharpe ratio which was crucial for this project. Going further, the calculations which are displayed as a pie chart are the best portfolio weightings expressed as percentages: how much of the investor's money should be put into each company. To do this, I calculated the best weights based on each Sharpe ratio for each ticker.
